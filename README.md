# Global-Superstore-Business-Performance-Power-BI-Dashborad

📊 Global Superstore Business Performance Dashboard

📌 Project Overview
This project is an interactive Business Performance Dashboard built using Power BI, based on the Global Superstore dataset. The dashboard provides key insights into sales, profit, orders, and customer behavior across different regions, segments, and categories.It helps stakeholders quickly understand business performance and make data-driven decisions.

🎯 Objectives
Analyze overall sales and profit performance
Track monthly trends of sales and profit
Understand performance by region, segment, and category
Identify top-performing sub-categories
Analyze shipping modes and order distribution

📈 Key Metrics (KPIs)
💰 Total Sales: 7.58M
📊 Total Profit: 890K+
📦 Total Orders: 30.78K
🔖 Average Discount: 0.14

📊 Dashboard Features
Sales by Segment (Consumer, Corporate, Home Office)
Sales by Region (Asia, Oceania, etc.)
Monthly Sales & Profit Trend Analysis
Sales by Category (Technology, Furniture, Office Supplies)
Sales by Sub-Category (Phones, Bookcases, Chairs, etc.)
Orders by Ship Mode

Interactive filters for:
Year (2011, 2012)
Ship Mode (First Class, Same Day, Standard Class)

🛠️ Tools & Technologies Used
Power BI
Data Visualization Techniques
Data Cleaning & Transformation (Power Query)
DAX (for calculated measures)

📌 Insights
Technology category generates the highest sales
Standard Class shipping is most frequently used
Sales and profit show a steady growth trend over months
Phones and Bookcases are top-performing sub-categories

🚀 How to Use
Download the .pbix file from this repository
Open it in Power BI Desktop
Interact with filters and visuals to explore insights
